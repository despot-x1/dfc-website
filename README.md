# 🍗 DFC — Danish Fried Chicken

> **Crispy. Juicy. Danish Style.**  
> A full-featured food ordering website built with pure HTML, CSS & JavaScript.

🌐 **Live Site:** [despot-x1.github.io/dfc-website](https://despot-x1.github.io/dfc-website)

---

## 📸 About

DFC is a modern, mobile-friendly restaurant website for Danish Fried Chicken — a fried chicken brand based in India. Customers can browse the menu, add items to cart, apply coupons, place orders, and track delivery in real time.

---

## ✨ Features

- 🏠 **Home Page** — Hero section, deals, testimonials & footer
- 🍗 **Full Menu** — Browse by category with search & filter
- 🛒 **Cart Sidebar** — Add/remove items, quantity controls, coupon codes
- 📦 **Checkout** — Delivery details + payment (COD / UPI / Card)
- ✅ **Payment Success** — Animated order confirmation with confetti
- 📍 **Order Tracking** — Live step-by-step delivery status
- 👤 **Auth** — Login & Sign Up with order history
- 🔔 **Product Popup** — Smooth slide-up modal when tapping any item
- 📱 **Fully Responsive** — Works on mobile, tablet & desktop

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure |
| CSS3 | Styling & animations |
| Vanilla JavaScript | All interactivity |
| Google Fonts | Barlow Condensed, Barlow, Playfair Display |
| Unsplash | Food photography |

> No frameworks. No dependencies. Pure HTML in a single file.
